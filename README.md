# 프로젝트 개요

## 프로젝트 소개
SSAL은 개발자들의 몸값이 비싼 요즘 준수한 실력의 동남아 개발자(베트남)들을 데려오면 어떨까? 하는 생각에서 시작한 프로젝트입니다.
한국어로 개발할 수 있는 수준의 한국어를 교육 시키기 위해 AI를 활용하여 듣기 / 말하기 등에 초점을 두어 국내 개발자들과 소통할 수 있게 하는 것을 목표로 하고 있습니다.
